Implementation Plan

Phased approach to implementing all design updates.

1

Card System Updates

Implement Option A card (left accent + prominent type badge) AssetCard.tsx
Type badge shows full text (no max-width truncation)
Create CompactCard component for list view CompactCard.tsx
Add view toggle (grid/list) to board pages
2

Horizontal Pills Navigation

Create HorizontalFilters component (Option A pills) HorizontalFilters.tsx
Replace FilterSidebar with horizontal pills
Update layout CSS (remove sidebar margin) globals.css
3

Search Experience

Enhance PageHeader search (larger, 640px max-width) PageHeader.tsx
Add type/tag results to search dropdown api/search/route.ts
Create tag results page library/tag/[slug]/page.tsx
4

Homepage with CMS

Create Homepage component with CMS sections app/page.tsx
Add API for homepage config api/homepage-config/route.ts
Create admin page for homepage management admin/homepage/page.tsx

  • components/library/HorizontalFilters.tsx - Option A pills nav
