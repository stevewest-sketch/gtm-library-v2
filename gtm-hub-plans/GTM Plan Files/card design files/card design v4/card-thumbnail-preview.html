Option Comparison

Option Height Pros Cons
A: Current 156px Clean, text-focused, more cards visible No visual preview of content
B: Top Thumb 220px Best visual impact, clear preview Taller cards = fewer visible, placeholder for non-Google
C: Side Thumb 156px Same height, preview visible Narrower text area, awkward crop on landscape content
D: Corner Thumb 156px Subtle, same height, optional Very small preview, may not be worth the complexity
Note: Thumbnails work automatically for Google Slides, Docs, Sheets, Drive files, and YouTube videos. For Figma, Canva, and other sources, a placeholder icon is shown instead (no public thumbnail API available).
